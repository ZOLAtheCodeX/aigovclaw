# MAINTAINER TOOLING

The skills in this directory are internal tooling for AIGovOps maintainers (Zola + Claude Code). They are not distributed as part of the AIGovClaw runtime and are not registered in the Hermes tool registry. End-users running AIGovClaw do not invoke, configure, or encounter these skills.

Category: same as CI workflows, dependabot configuration, pre-commit hooks. Infrastructure for repository maintenance.

Scope decision locked on 2026-04-18. Do not promote these skills to product features. Do not expose them via the runtime. If the scope changes, update this banner and the `jules/README.md` section explicitly.

For the product-facing side of AIGovClaw, see the repo root `README.md` and `claw/`.
